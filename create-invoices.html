1
Enter your Google API Key and OAuth Client ID (same ones used on the Work Order page), then click Connect. The page will sign you in and load all docs from the x -TestInvoice folder in your Google Drive.
2
For QuickBooks, go to developer.intuit.com, create an app, and copy your Client ID and Client Secret into the QuickBooks section below. Set the redirect URI to this page's URL.
3
Click Connect QuickBooks to authorise. Once connected, select the work order docs to invoice and click Create Selected Invoices.
💡
Each checked work order doc is read, key billing fields are extracted (client, amount, line items), and a draft invoice is created in QuickBooks under the matching customer.
